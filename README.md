## README

- `data/`: contém os dados usados no projeto, como arquivos brutos e tratados.
-   `data_raw/`: dados brutos.
-   `data_interim/`: dados intermediários.
-   `data_processed/`: dados processados.    
- `documentation/`: documentações e referências eventualmente necessárias.
- `reports/`: scripts e visualizações para reportar os resultados.
- `scripts/`: guarda o código-fonte principal do projeto.
